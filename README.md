keitetsu's Dotfiles
===================

## Installation

```
git clone git@github.com:keitetsu/dotfiles.git "$HOME/.config/keitetsu"
bash "$HOME/.config/keitetsu/install.sh"
```

## User configuration

The directory `$HOME/.config/keitetsu/user` can be created for further
user-defined configuration. The following files are loaded
automatically, if they exist:

* bashrc
* emacs/custom.el
* emacs/init.el
* gitconfig
* profile
